# Iris Data Visualization

## Problem Statement

The objective of this project is to analyze and visualize the Iris dataset using different graphical techniques to understand patterns and relationships among flower species.

## Dataset

Iris Dataset from Kaggle.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Visualizations Created

### Bar Chart

Visualized the distribution of flower species in the dataset.

### Histogram

Analyzed the distribution of sepal length values.

### Scatter Plot

Compared sepal length and petal length across different species.

## Observations

* All three species are represented in the dataset.
* Most flowers have sepal lengths between 5 and 7 cm.
* Sepal length and petal length show a positive relationship.
* Different species form distinct clusters in the scatter plot.

## Conclusion

The project successfully visualized the Iris dataset using multiple chart types and identified important patterns among flower species.

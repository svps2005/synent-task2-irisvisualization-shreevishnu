# Iris Data Visualization Report

## Problem Statement

The objective of this project was to visualize and analyze the Iris dataset using different graphical techniques to understand patterns and relationships among flower species.

## Dataset

Iris Dataset from Kaggle.

## Methodology

1. Loaded the Iris dataset using Pandas.
2. Explored the dataset structure and features.
3. Created a Bar Chart to visualize species distribution.
4. Created a Histogram to analyze the distribution of sepal length.
5. Created a Scatter Plot to compare sepal length and petal length.
6. Interpreted patterns and relationships observed in the visualizations.

## Visualizations

### Bar Chart

Displayed the number of flowers belonging to each species.

### Histogram

Showed the frequency distribution of sepal length values.

### Scatter Plot

Compared sepal length and petal length for different flower species.

## Results

* All three Iris species were represented in the dataset.
* Most flowers had sepal lengths between 5 cm and 7 cm.
* A positive relationship was observed between sepal length and petal length.
* Different species formed distinct clusters in the scatter plot.

## Conclusion

Data visualization helped identify important patterns and feature relationships within the Iris dataset. The project demonstrated how graphical analysis can provide meaningful insights from data.
